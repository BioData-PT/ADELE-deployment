How to verify the checksum
After downloading TRE.pub, run the following command in your terminal and compare the output to the checksum above:
  • Linux sha256sum TRE.pub
  • macOS shasum -a 256 TRE.pub
  • Windows (PowerShell) (Get-FileHash .\TRE.pub -Algorithm SHA256).Hash.ToLower()
If the checksum matches, your download is correct.

About Crypt4GH

Crypt4GH is a secure file format and encryption tool for genomic data, created by GA4GH. It encrypts files so only authorized recipients can decrypt them.

See the official Crypt4GH repository.

Troubleshooting

  • Windows: In PowerShell, use the pipeline method shown above; redirection </> may corrupt binary data.
  • “command not found”: Ensure crypt4gh is on your PATH (restart terminal after install, or use pipx).
  • Wrong key type: The TRE key must be a Crypt4GH public key, not an SSH/PGP key.
